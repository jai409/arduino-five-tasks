# Five Tasks. One Board. 🚀

> **Arduino Lab Exercises — Marcus Jaijal · RNU University**

A comprehensive, beautifully designed web guide featuring five foundational embedded systems exercises using an Arduino Uno. This project provides full Arduino C++ code, component lists, and step-by-step wiring guides for each task.

🌍 **[View the Live Guide](https://jai409.github.io/arduino-five-tasks/)**

---

## 📋 The Tasks

This guide covers real-world use cases and core concepts of embedded engineering, progressing from simple digital outputs to complex I2C display systems:

### 01 · Basic Output Control
🔴 **LED · Digital Output**
Control an LED using a digital output pin. The foundation of all embedded output logic (e.g., traffic lights, status indicators).

### 02 · Variable Control System
💡 **PWM · Brightness Fade**
Simulate a fade effect through variable voltage output using Pulse Width Modulation (PWM) (e.g., smart lighting, motor speed control).

### 03 · User Interaction System
🔘 **Button · Digital Input**
Read digital input and manage state by toggling an LED on and off with a push button, including debounce logic (e.g., elevator buttons, manual overrides).

### 04 · Alert / Notification System
🔔 **Buzzer · Tone Output**
Trigger ascending alarm tones on a passive buzzer using a button (e.g., fire alarms, security systems).

### 05 · Display / Information System
📟 **DHT11 · LCD I2C**
A complete sensor-to-display pipeline. Read live temperature and humidity data from a DHT11 sensor and display it on a 16x2 I2C LCD (e.g., weather stations, smart home dashboards).
